# Design handoff: redo the notification fold-wheel icon

For: Claude Design session
From: the session that built the Android notification panel (branch
`claude/fold-tracking-notifications-panel-bhjcxd`)

## What this artwork is

The Sourdough Timer app pins a persistent card in the Android pull-down
notification shade during bulk fermentation. The card's large icon is a
"fold progress wheel": a honey ring that fills clockwise as the baker
records stretch-and-folds, wrapped around a small illustration of
fermenting dough. There are **13 frames** (ring at 0/12 … 12/12), and the
Android code picks the frame closest to `completedFolds / plannedFolds`.

- Frames live at `assets/images/fold-wheel/wheel-{0..12}.png`
- Generated by `node tools/generate_notification_icons.mjs` — a
  zero-dependency Node script that draws with signed-distance functions
  and writes PNGs via built-in zlib. Extend it, rewrite it, or produce the
  PNGs some other way — the only contract is the 13 output files.
- Consumed by `lib/bulkStatusPanel.android.ts` (`WHEEL_FRAMES` /
  `wheelFrame()`); no code changes needed if filenames stay the same.

## The problem with the current icon

The current center illustration is a smooth cream dome with bubbles
floating **above/outside** it. The owner's critique (correct):

1. It reads as a **shaped boule** — but during bulk fermentation the dough
   is *not* shaped yet; it's a loose, slack mass in a container.
2. The CO₂ bubbles would be **inside** the dough, not drifting outside it.

So the image is at odds with the exact stage of baking it represents.

## Concept direction from the owner

First concept to try: a **Cambro-style container** (straight-sided,
translucent dough tub — every serious home baker has one) **half filled
with loose dough** that has a **slightly domed top**, with **fermentation
microbes/bubbles visible inside the dough mass**. Because the tub is
translucent, showing the interior is naturally justified.

**Also explore a few other concepts** — the owner explicitly wants
alternatives to compare, not just the Cambro. Ideas worth sketching (feel
free to add your own):

- A cross-section / cutaway of the rising dough mass showing an open,
  bubbly interior (crumb-in-progress).
- A top-down look into a bowl of slack dough with surface bubbles just
  breaking through (what a baker actually sees mid-bulk).
- A glass jar with a rubber-band/measurement line and the dough risen past
  it — the classic "how far has it risen" cue.

Present the concepts as rendered frames (at least frame 8-of-12 and
12-of-12 per concept) so the owner can pick by looking, not by reading
descriptions. The owner is non-technical — show real output.

## Hard constraints

- **13 PNGs**, 384×384, transparent background, named `wheel-0.png` …
  `wheel-12.png`, where the number is the filled twelfths of the progress
  ring. Frame 12 (all folds done) stays visible for hours — it should look
  serenely complete, not alarming.
- **Keep the progress ring** — it *is* the fold-progress wheel the feature
  promises. Current spec: honey arc (`#E8A33D`), faint cream track,
  rounded caps, sweeping clockwise from 12 o'clock. Restyling is fine;
  removing it is not.
- **It renders tiny.** The notification largeIcon slot is roughly
  40–64 dp, and some Android skins crop it to a circle. Silhouettes must
  survive at ~48 px: bold shapes, few elements, strong value contrast.
  Check every concept by viewing the PNG scaled down to ~48 px.
- **Palette:** stay on the app's warm artisan-bakery palette in
  `components/theme.ts` — espresso `#171210`, cream `#F2E8DC`, honey
  accent `#E8A33D`, plus the animation's colors (below). Calm and warm,
  never techy/neon.
- Works on any shade background (light and dark system themes) — hence the
  dark disc behind everything in the current version; keep some solid
  backing shape so the art never sits raw on white.

## Visual reference

The icon should feel like a still from the app's live fermentation
animation, `components/SkiaFermentationScene.tsx`: luminous **amber yeast
cells**, **violet LAB rod-chains** (`#c084fc` family), rising **CO₂
bubbles** with bright rims and specular dots, all glowing softly against
near-black. "Microbes inside the dough" can borrow those exact organisms
so the notification and the in-app scene read as one world.
