# References: Bread Science & Fermentation

PDFs in this folder are the canonical sources for fermentation algorithms, rise modeling, temperature corrections, and diagnostic advice in the app.

- **Bread Science** (Emily Buehler) — Q10 kinetics, yeast metabolism, dough structure
- **Modernist Bread** (Myhrvold & Migoya) — fermentation biochemistry, temperature tables
- **The Rye Baker** (Stanley Ginsberg) — long fermentation, wild yeast & bacteria behavior
- **The Sourdough School** (Vanessa Kimbell) — microbiome, starter strength, fermentation markers

## Primary Research

- [The Q10 Rule in Bread Fermentation](https://en.wikipedia.org/wiki/Q10_(temperature_coefficient)) — ~2× fermentation rate per 15°F
- [Wild Yeast & LAB Kinetics in Sourdough](https://www.researchgate.net/publication/268555098_The_microbiology_of_sourdough) — starter population dynamics
